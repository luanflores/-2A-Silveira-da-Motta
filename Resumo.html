Front-end

  • A tag (HTML): Estrutura do nosso documento antes de começar a colocar o conteúdo inserido uma tag html. Dentro da tag é necessário declarar outras duas tags (head) e (body). Estas duas tags são "irmãs", pois estão no mesmo nivel hierarquico em relação a sua tag "mãe" que é (HTML).
  • Tag (HEAD): contém informações sobre o documento html, que são de interrese somente do navegador e para outros serviços da web, são informações que podemos declarar os metadados sobre a página.
  • Tag (h1)-(h6): Elementos html de cabeçalho. Representam seus elementos html de título (h1) é o nivel mais alto e (h6) o mais baixo.
  • Tag (p): Elemento de parágrafo. Representa um parágrafo em medidas visuais, parágrafos são representados como bloco de indentação de texto.
  • Get e Post: Get é aquele que envia os parâmetros na url, tem tamanho limitado e geralmente usado para consulta. Post é aquele que envia os parâmetros na requisição e não tem limite no tamanho
  • Reset: São baseados em recursos, que são representados como url, e são acessados usando métodos http, como get, post, put e delet.
  • Tag (CSS): Define a aparência de uma página da web que permite aplicar cores, fontes espaçamentos, tamanhos, posições e muitos outros elementosde design ao conteúdo html. Linguagem html e css são de marcação e não de programação. A cor pode ser representada como "color" para especificar a cor de fundo de um elemento ou "background-color", as cores no css podem ser especificadas pelo nome, código rgb, código hex em palavra chave, usado como valor de propriedade de estilo. 0.0 a 1.0 as cores sao rgb(red, green, blue)
  • Código hex: apresenta a cor como uma combinação de seis dígitos como "#FF0000" para vermelho, "#0000FF" para o azul.
  • Tag IMG: Especifica a url da imagem a ser exibida, a largura, altura, descrição da imagem entre outros.
  • UL type="a": Especificação tipo de marcador sendo letras, números e números romanos.
